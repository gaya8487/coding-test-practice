## 투 포인터 
O(nlongn)의 시간 복잡도를 가짐

포인트 두개를 사용하여 탐색한다.

### 투 포인터의 이동원칙
포인트의 위치를 배열의 첫번째(i), 마지막에 놓고(j) 전체 배열(A)을 검색한다고 했을 때
투 포인터의 이동원칙은 다음과 같다

```java
A[i]+A[j] > 찾으려는 숫자 : j--;
A[i]+A[j] < 찾으려는 숫자 : i--;
A[i]+A[j] == M : i++; j--; count ++;
```


### 구간 합을 구하는 공식
```java
while (i<j){
	if(A[i]+A[j] > 찾으려는 숫자 ){
		j-- 
	}else if(A[i]+A[j] < 찾으려는 숫자 ){
		i++
	}else{
		i++;
		j--;
		count++;
    }
}
```
